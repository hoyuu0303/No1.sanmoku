#説明書
- **No1.py**をひらいてコードを**コピー**する。
- Visual studio codeを開く。
- Visual studio code内でファイル＞新しいテキストファイルへいってクリックする。
- コピーしたコードをVisual studio code内に**貼り付け**る。
-  画面右上に**三角マーク**のボタンが表示されているため、クリックする。
-  三目並べができる。
-  縦横斜め同じ記号を表示させたら勝ち、一人プレイ用なので自分で○×を交互に押します。
